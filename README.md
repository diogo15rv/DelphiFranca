# Projeto Delphi para a equipe França
Teste desenvolvido para a empresa SoftPlan

Projeto desenvolvido de acordo com as solicitações e tentando extrair o máximo em padrões de projeto.
Os testes também estão disponibilizados no diretório do projeto.

Não foi utilizado nenhum tipo de componentes de terceiros, e para realizar a demanda 2 em Thread foi utilizada a classe Thread e não Tasks(caso preferível em task, por gentileza solicitar que realizo a implementação do mesmo).

Para compilar corretamente o projeto, deve-se instalar o componente "spComponentes" primeiramente, para depois compilar sem erros o ProvaDelphiApp.

Executáveis do projeto está disponíveis no diretório .\Bin
